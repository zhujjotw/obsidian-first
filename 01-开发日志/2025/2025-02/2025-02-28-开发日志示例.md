**要点**：
- 单一类负责所有业务逻辑
- 80+ 个 getter 方法提供数据访问
- 与 PLMClient、FTPClient 等外部系统解耦

**代码结构**:
```python
class BaseService:
    def __init__(self, owner, project_name, ...):
        self.plm_client = PLMClient()
        self.ftp_client = FTPClient()

    # 项目验证
    def is_project_exist(self) -> bool:
        pass

    # 数据获取
    def get_test_request_version(self) -> str:
        pass
```
