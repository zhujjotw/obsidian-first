## 📝 最新日志

### 本周

```dataview
TABLE without id
  file.link as "日期",
  project as "项目",
  category as "类型",
  status as "状态"
FROM #开发日志
WHERE dateformat(date, "yyyy-WW") = dateformat(date(today), "yyyy-WW")
```

## 🐛 问题追踪

### 待处理

```dataview
TABLE without id
  file.link as "问题",
  priority as "优先级",
  dateformat(created, "MM-dd") as "创建日期"
FROM #问题
WHERE status = "待处理" OR status = "处理中"
SORT priority DESC, created DESC
```

### 最近解决

```dataview
TABLE without id
  file.link as "问题",
  dateformat(created, "MM-dd") as "创建",
  dateformat(date, "MM-dd") as "解决"
FROM #问题
WHERE status = "已解决"
SORT created DESC
LIMIT 10
```

## 🏷️ 标签云

```dataview

  rows.file.link as "数量"
FROM #开发日志

GROUP BY tag
SORT length(rows) DESC
```

## ⚙️ 使用指南

### 创建新日志
1. 使用 `Ctrl/Cmd + N` 创建新笔记
2. 输入 `/devlog` 插入日志模板
3. 填写内容

### 记录问题
1. 使用 `/issue` 插入问题模板
2. 填写问题描述和解决方案

### 添加代码片段
1. 使用 `/snippet` 插入代码模板
2. 添加代码和说明
