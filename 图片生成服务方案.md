# 图片生成服务方案

## 需求概述
- 输入指令生成旗帜图（根据规定尺寸）
- 输入指令生成背景图（契合场景的效果图）
- 指令简单易用
- 后端：zimage（已部署）
- 前端：H5

## API 设计

### 生成图片接口
```
POST /api/image/generate

请求体:
{
  "type": "flag" | "background",
  "prompt": "用户输入的指令",
  "params": {
    "width": 1920,      // 可选，默认值
    "height": 1080,     // 可选，默认值
    "style": "realistic", // 可选
    "format": "png"     // 可选
  }
}

响应:
{
  "code": 0,
  "data": {
    "imageUrl": "https://cdn.example.com/xxx.png",
    "taskId": "task_123456",
    "width": 1920,
    "height": 1080
  }
}
```

### 预设场景列表接口
```
GET /api/scenes?type=flag|background

响应:
{
  "code": 0,
  "data": [
    {"id": "festival", "name": "节日", "icon": "..."},
    {"id": "corporate", "name": "企业", "icon": "..."},
    ...
  ]
}
```
