sed -i 's/:3/:5/g' /var/local/hme_config.xml;  
sed -i 's/YOUTUBE_SET_URL_TEST=0/YOUTUBE_SET_URL_TEST=1/g' /opt/youtube/status.ini;  
sed -i 's/YOUTUBE_ENABLE_PRELOAD=1/YOUTUBE_ENABLE_PRELOAD=0/g' /opt/youtube/status.ini;

### MM/MI 日志抓取流程

1. 启动日志：
    
    setenforce 0  
    cd /vendor/bin/autolog/modules/platform/mm  
    sh mmdbg_on.sh /OAD/
    
2. 修改模块配置：
    
    mount -o remount rw /vendor/tvconfig/  
    chmod 777 /vendor/tvconfig/bsp/board/BD_MT5873_H1V1_B6_S/module/MStar_Default_Module.ini  
    ./mi_mm_debug_on.sh -i /vendor/tvconfig/bsp/board/BD_MT5873_H1V1_B6_S/module/MStar_Default_Module.ini /OAD/  
    sync  
    reboot
    
3. 复现问题后上传日志：
    
    /OAD/hui_streaminject_log  
    /OAD/MM_RAM*  
    /OAD/MI_*

### AML950 固件烧录步骤

1. 将 `aml_upgrade_package.img` 拷贝到 U盘根目录；
    
2. 将 `aml_sdc_burn.ini` 放入 U盘根目录；
    
3. 串口波特率设置为 921600；
    
4. 按住电脑回车键给电视板上电，进入 U-boot；
    
5. 输入命令开始烧录：
    
    usb_burn aml_sdc_burn.ini
