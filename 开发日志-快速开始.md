# 开发日志知识库 - 快速开始

## ⚡ 5分钟上手

### 第一步：安装插件

在 Obsidian 中打开 **设置 → 社区插件**，安装并启用：

| 插件 | 用途 | 必需 |
|------|------|------|
| [Dataview](https://github.com/blacksmithgu/obsidian-dataview) | 数据查询 | ✅ |
| [Templates](https://help.obsidian.md/Extending+Obsidian/Templates) | 模板功能 | ✅ |
| [Tasks](https://github.com/obsidian-tasks-group/obsidian-tasks) | 任务管理 | ⭐ |
| [Calendar](https://github.com/liamcain/obsidian-calendar-plugin) | 日历视图 | ⭐ |

---

### 第二步：配置模板

1. 设置 → 核心插件 → 模板
2. 模板文件夹位置：`Templates/`
3. 日期格式：`YYYY-MM-DD HH:mm`

---

### 第三步：创建第一条日志

```
1. Ctrl/Cmd + P (命令面板)
2. 输入 "插入模板"
3. 选择 "开发日志模板"
4. 填写内容
```

## 🎯 核心模板

### 1. 开发日志

**用途**：记录每日工作

**快捷键**：`/devlog` (需配置)

**关键字段**：
```yaml
---
type: devlog
date: 2025-02-28
project: 项目名
category: 功能开发
status: 已完成
---
```

### 2. 问题记录

**用途**：追踪 Bug

**关键字段**：
```yaml
---
type: issue
status: 待处理/处理中/已解决
priority: 高/中/低
---
```

### 3. 代码片段

**用途**：收集代码

**关键字段**：
```yaml
---
type: snippet
language: python
---
```

## 🔍 常用查询

### 查看今天的日志
```dataview
LIST
FROM #开发日志
WHERE date = date(today)
```

### 查看本周日志
```dataview
TABLE file.link as "日期", project as "项目"
FROM #开发日志
WHERE dateformat(date, "yyyy-WW") = dateformat(date(today), "yyyy-WW")
SORT date DESC
```

### 查看待解决问题
```dataview
TASK
WHERE !completed
FROM #问题
GROUP BY priority
```

## 💡 最佳实践

### 1. 保持简洁
- 日志不需要太详细
- 重点记录关键决策和问题
- 使用代码链接而非复制代码

### 2. 建立关联
- 使用 `[[链接]]` 关联相关笔记
- 添加标签便于查询
- 及时更新索引页面

### 3. 定期回顾
- 每周五回顾本周日志
- 每月做月度总结
- 提取有价值的内容到知识库

## 🚀 进阶技巧

### 双向链接

```markdown
<!-- 链接到项目文档 -->
今天完成了 [[项目A]] 的功能

<!-- 链接到问题 -->
解决了 [[问题-登录超时]]

<!-- 链接到代码 -->
参考 [[Python/装饰器]]

<!-- 嵌入内容 -->
![[项目A#架构图]]
```

### Dataview 高级查询

```dataview
TABLE without id
  file.link as "日志",
  project as "项目",
  length(filter(rows.file.link, (f) => contains(f.tags, "已完成"))) as "完成数"
FROM #开发日志
WHERE dateformat(date, "yyyy-MM") = dateformat(date(today), "yyyy-MM")
GROUP BY date, project
```

### 快捷命令（QuickAdd）

创建每日日志的宏：
1. 创建新文件
2. 自动命名：`开发日志-YYYY-MM-DD`
3. 插入模板
4. 移动到对应月份文件夹

## ❓ 常见问题

**Q: 如何快速创建日志？**
A: 安装 QuickAdd 插件，设置快捷键或命令

**Q: 如何统计项目工作量？**
A: 使用 Dataview 按项目分组统计日志数量

**Q: 如何与团队共享？**
A: 使用 Obsidian Sync 或 Git 同步笔记库

**Q: 移动端如何使用？**
A: iOS 用快捷指令，Android 用 Tasker
