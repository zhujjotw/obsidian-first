# Obsidian 代码与文档集成模板

这套模板旨在帮助你在 Obsidian 中完美集成代码编写和文档管理，实现技术写作与代码开发的无缝衔接。

## 🎯 模板体系

### 1. 代码执行模板 (`代码执行模板.md`)
**用途**: 创建可执行的代码示例文档
**特点**:
- 支持多种编程语言
- 集成实时代码执行
- 自动关联相关资源
- 包含代码解析和说明

**使用场景**:
- 技术教程编写
- 代码示例分享
- API文档生成
- 学习笔记制作

### 2. 项目文档模板 (`项目文档模板.md`)
**用途**: 完整的项目技术文档
**特点**:
- 项目概览和技术栈管理
- 代码结构可视化
- API文档自动生成
- 进度和问题跟踪

**使用场景**:
- 项目启动和规划
- 技术架构文档
- 开发进度跟踪
- 团队协作

### 3. 教程文档模板 (`教程文档模板.md`)
**用途**: 系统性的学习教程
**特点**:
- 标准化的教学结构
- 步骤化实践指导
- 常见问题解答
- 进阶资源推荐

**使用场景**:
- 新手入门教程
- 技能提升指南
- 在线课程制作
- 团队培训

## 🔧 核心功能

### 代码执行集成
```markdown
// 支持实时代码执行
```javascript
console.log("Hello, World!");
```

// 使用 Code Runner 插件一键执行
```

### 动态数据展示
```dataview
// 自动关联相关代码片段
TABLE file.link as "代码片段", language as "语言", created as "创建时间"
FROM #代码片段
WHERE contains(tags, "JavaScript")
SORT created DESC
LIMIT 5
```

## 🚀 快速开始

### 1. 安装必要插件
```bash
# 在 Obsidian 社区插件中安装：
- Code Runner      # 代码执行
- Templater        # 模板自动化
- Dataview         # 数据查询
- Excalidraw       # 图表绘制
```

### 2. 设置模板
1. 复制模板文件到你的项目目录
2. 根据需要修改模板内容
3. 使用 Templater 插件快速创建新文档

### 3. 建立工作流程
```
项目规划 → 代码编写 → 文档记录 → 同步更新
    ↓         ↓         ↓         ↓
  项目模板 → 代码片段 → 教程文档 → 自动化同步
```

## 🎨 高级技巧

### 模板自动化
```javascript
// 使用 Templater 创建高级模板
<%*
let title = await tp.system.prompt("文档标题");
let project = await tp.system.prompt("项目名称");
tR += `# ${title}\n\n`;
tR += `项目: ${project}\n\n`;
%>
```

### 动态内容生成
```dataview
// 自动生成项目目录
TABLE WITHOUT ID
  file.link as "文档",
  type as "类型",
  created as "创建时间"
FROM #项目
WHERE project = "{{project_name}}"
SORT created DESC
```
