# 开发日志知识库使用指南

## 🚀 快速开始

### 第一步：安装必要插件

1. **Dataview** - 数据查询（必需）
2. **Templates** - 模板功能（必需）
3. **Calendar** - 日历视图（推荐）
4. **Tasks** - 任务管理（推荐）
5. **QuickAdd** - 快速添加（推荐）

### 第二步：配置模板文件夹

1. 设置 → 核心插件 → 模板
2. 指定模板文件夹为：`Templates/`
3. 设置模板日期格式：`YYYY-MM-DD HH:mm`

### 第三步：开始记录

- 按 `Ctrl/Cmd + P` 输入 "插入模板"
- 选择对应的模板
- 填写内容

## 📝 核心功能

### 1. 开发日志 (DevLog)

**用途**：记录每日工作内容

**使用方法**：
```
Ctrl/Cmd + P → 插入模板 → 开发日志模板
```

**关键字段**：
- `type: devlog` - 标识为开发日志
- `date:` - 日志日期
- `project:` - 所属项目
- `tags:` - 标签（自动按月分类）

### 2. 问题记录 (Issue)

**用途**：追踪Bug和问题

**使用方法**：
```
Ctrl/Cmd + P → 插入模板 → 问题记录模板
```

**状态流转**：
```
待处理 → 处理中 → 已解决/无法解决
```

### 3. 代码片段 (Snippet)

**用途**：收集有用代码

**使用方法**：
```
Ctrl/Cmd + P → 插入模板 → 代码片段模板
```

## 🔍 查询方法

### Dataview 基础查询

#### 查找所有开发日志
```dataview
LIST
FROM #开发日志
SORT date DESC
```

#### 查找特定项目的日志
```dataview
LIST
FROM #开发日志
WHERE project = "转生产设计联系书创建工具"
SORT date DESC
```

#### 查找本周日志
```dataview
TABLE without id
  file.link as "日期",
  project as "项目",
  category as "类型"
FROM #开发日志
WHERE dateformat(date, "yyyy-WW") = dateformat(date(today), "yyyy-WW")
SORT date DESC
```

#### 查找未解决的问题
```dataview
TASK
WHERE !completed
FROM #问题
GROUP BY priority
```

### 高级查询

#### 按时间段查询
```dataview
LIST
FROM #开发日志
WHERE date >= date(today) - dur(7 days)
SORT date DESC
```

#### 统计项目工作量
```dataview
TABLE without id
  project as "项目",
  length(rows.file.link) as "日志数"
FROM #开发日志
WHERE dateformat(date, "yyyy-MM") = dateformat(date(today), "yyyy-MM")
GROUP BY project
```

## 📊 数据分析

### 每日工作统计

```dataview
TABLE rows.file.link as "日志"
FROM #开发日志
WHERE date = date({{today}})
GROUP BY date
```

### 项目时间分布

```dataview
PIE project
FROM #开发日志
WHERE dateformat(date, "yyyy-MM") = dateformat(date(today), "yyyy-MM")
GROUP BY project
```

### 问题解决率

```dataview
CALCULATE sum(1) FROM #问题 WHERE status = "已解决"
/
CALCULATE sum(1) FROM #问题
* 100
```

## ⚙️ 自动化

### 使用 QuickAdd 创建宏

**每日日志宏**：
1. 创建新文件
2. 插入开发日志模板
3. 自动填充日期
4. 文件名格式：`开发日志-YYYY-MM-DD`

### 使用 Templater 高级功能

```javascript
// Templater 模板示例
<%*
let date = tp.date.now("YYYY-MM-DD");
let title = await tp.system.prompt("日志标题");
tR += `# ${date} - ${title}\n\n`;
tR += `## 今日目标\n\n`;
tR += `- [ ] \n`;
%>
```
